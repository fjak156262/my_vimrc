# Prerequisite 
**```GIT```**

**```CURL```**

```YARN```

```NODEJS```






# INSTALLATION
**FOLLOW THESE STEPS BELOW**

1. ``` git clone https://github.com/fjak156262/my_vimrc/ ```

2. ``` cd my_vimrc ```

3. ``` cp .vimrc $HOME ```

4. ``` vi $HOME/.vimrc ```

***IF IT IS NOT WORKING THEN RUN ```:PlugInstall``` MANUALLY .vimrc FILE***
